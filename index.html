Core Techniques

Built on Responsive Principles

Every technique used here maps directly to the Project 2 implementation checklist from DecodeLabs.

📐

CSS Grid (Macro)

The outer scaffold — header, sidebar, main, footer. Grid defines the structural architecture, giving each zone its rightful space.

🧩

Flexbox (Micro)

Inside every card and component, Flexbox aligns, distributes, and wraps content fluidly — the furniture inside the house.

📱

Mobile-First CSS

Base styles target the smallest screen. @media (min-width) progressively layers desktop enhancements on top.

🔤

Fluid Typography

Every font size uses clamp(min, fluid, max) — text scales smoothly across the full viewport range with no abrupt jumps.

☰

Popover Navigation

The hamburger menu uses the native HTML Popover API. No JavaScript required — built-in accessibility and backdrop included.

♿

Accessible Targets

All interactive elements meet the 44×44px minimum touch target size. Zoom is never blocked — user-scalable=no is forbidden.
